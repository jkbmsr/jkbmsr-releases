JKBMSR Firmware

Flash your ESP32 gateway from Chrome or Edge

Install firmware over USB with the Web Serial API — no drivers or Arduino IDE. Desktop Chrome or Edge only (not Firefox, Safari, or mobile).

1
Flash Install the latest esp32dev build (loading…).
2
Connect Wi‑Fi After install, the same dialog offers Wi‑Fi setup via Improv Serial (SSID + password). Keep the USB cable plugged in.
3
Claim the device On success you’re sent to app.jkbmsr.com/onboard with the device ID and claim code filled in — sign in and confirm.

Before you start

  1. Use a USB data cable (not charge-only).
  2. Close Arduino IDE, PlatformIO monitor, or anything else holding the serial port.
  3. Click install, pick the ESP32 port when Chrome prompts you.
  4. Don’t unplug while flashing. After flash, leave USB connected for Wi‑Fi setup.
Your browser doesn't support Web Serial. Use desktop Chrome or Edge — this won't work on mobile or in Firefox/Safari. This page needs HTTPS (or localhost) for Web Serial. On production it should be cdn.jkbmsr.com.

Field fallback

No USB later? Hold the ESP32 BOOT button ~5 seconds to re-enter setup, join the SoftAP JKBMSR-Setup-XXXX (password = claim code on the label / serial log), and submit Wi‑Fi from your phone.

This CDN build predates Wi‑Fi provisioning

Public release does not yet include Improv / SoftAP setup. After flashing it, the device will sit offline until you install a provisioning-capable build (firmware 0.1.4+) via a new release or a local pio run -e dev -t upload. The install button still works for a baseline flash.
