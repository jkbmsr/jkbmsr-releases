JKBMSR Firmware

Flash your ESP32 gateway from Chrome or Edge

This installs the latest firmware for esp32dev (currently loading…) directly over USB using the Web Serial API — no software to install. Works in desktop Chrome or Edge only; Firefox and Safari, and all mobile browsers, do not support Web Serial.

Before you start

  1. Connect the ESP32 to this computer with a USB data cable (not charge-only).
  2. Close any other program using the serial port (Arduino IDE, PlatformIO monitor, etc.).
  3. Click install below, then pick the ESP32's port when Chrome prompts you.
  4. Do not disconnect the device while flashing is in progress.
Your browser doesn't support Web Serial. Use desktop Chrome or Edge — this won't work on mobile or in Firefox/Safari. This page needs to be served over HTTPS (or localhost) to flash a device — you're seeing this in error if you're on cdn.jkbmsr.com.
Flashing alone won't get the device online yet

jkbmsr-firmware doesn't have a Wi-Fi provisioning flow built yet (no captive portal, no BLE setup). A freshly flashed device boots with no stored Wi-Fi credentials and will sit idle — it won't connect to the cloud or show up in the dashboard on its own. If you need it online today, ask about a serial-based provisioning stopgap, or flash a build with credentials compiled in for this one test.
